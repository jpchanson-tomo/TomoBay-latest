This class is responsible for sending creating email and mail server objects. At the moment there is only a MailServerSend class but at some point it may be worth implementing a recieving interface as well.

In order to create email and server objects an AbstractEmailBuilder must be passed in. This builder determines the specifics of the email server that the system is to connect to.
