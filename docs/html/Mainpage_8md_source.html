1 Introduction{#mainpage}
2 =======================
3 \image html logoicon.png
4 
5 Welcome To TomoBay, the eBay management system, currently in its early stages. Core functionality is
6 in place but there is always room for extra bits and bobs.
7 
8 Features
9 --------
10 - Invoice generation through integration with '*Winstock*'
11 - Reverse Tax calculation from eBay total, based on *Winstock* unit prices
12 - eBay information retrieval: orders, items, teransactions, Customers etc
13 - Post Office DMO integration through CSV generation
14 - Local Storage using MySQL database and connection pooling
15 - Https Web frontend with a mix of static pages and AJAX using JQuery
16 - Thread Pooling
17 - Logging to a series of log files specific to a certain areas
18 - Built in Email client which auto mails certain people given certain actions
19 - Pause/Resume server functionality
